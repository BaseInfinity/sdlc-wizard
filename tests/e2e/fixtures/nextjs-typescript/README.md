# Next.js TypeScript Fixture

E2E test fixture representing a modern SaaS stack.

## Stack
- Next.js 14 (App Router)
- TypeScript
- Prisma (SQLite for testing)
- Jest + React Testing Library

## Commands
- `npm run dev` - Start dev server
- `npm run build` - Production build
- `npm run test` - Run tests
- `npx prisma migrate dev` - Run migrations

## Testing Scenarios
- Add new API route
- Add Prisma model
- Add React component with tests
