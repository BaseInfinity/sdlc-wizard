# MERN Stack Fixture

E2E test fixture representing a classic full-stack JavaScript application.

## Stack
- MongoDB (mocked for testing)
- Express.js
- React
- Node.js

## Structure
```
mern-stack/
├── server/     # Express API
│   ├── index.js
│   └── tests/
└── client/     # React app
    └── src/
```

## Commands
- Server: `cd server && npm run dev`
- Client: `cd client && npm start`
- Tests: `cd server && npm test`

## Testing Scenarios
- Add new API endpoint
- Add React component
- Add form validation
